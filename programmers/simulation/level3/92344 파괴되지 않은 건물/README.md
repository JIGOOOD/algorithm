## 💡 접근 및 설계

### 2d difference array

(r1, c1) ~ (r2, c2)에 특정 값 v를 더하려고 한다면,

1. 0으로 초기화된 (n+1) \* (m+1) diff 배열을 만든다. (꼭 0으로만 이루어져 있어야 함)
2. diff[r1][c1] += v

   diff[r1][c2+1] -= v

   diff[r2+1][c1] -= v

   diff[r2+1][c2+1] += v

   이렇게 네 점만 표시를 한다.

3. 가로 누적합을 한다.
4. 세로 누적합을 한다.
5. 그리고 원본 배열과 diff 배열을 합하면, 특정 값을 더한 배열이 나온다.

-> 시간 복잡도는 O(nm + q)이다. 이때, q는 특정 값 v를 더한 연산 횟수이다.
