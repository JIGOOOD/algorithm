### 난이도

★ 힌트나 답을 보고 쉽게 풀 수 있음
아이디어가 잘 떠오르진 않았음, 근데 답보니 알고리즘은 쉬움

### 알고리즘 분류

- BFS/DFS

### 핵심 아이디어

1. 모든 간선을 다 방문해야 함 = 오일러 경로 + Hierholzer
2. 인접 리스트를 사전순 최소 힙으로 구성
3. DFS로 갈 수 있는 곳까지 가다가, 막히면 결과에 push
   - 막힌 노드는 경로의 끝부분에 위치 → 뒤에서부터 채워나감
   - 진행하면서 push하면, 사전순 욕심내다가 막다른 길에 빠졌을 때 답이 망가짐
4. 마지막에 결과 배열을 뒤집으면 정답
