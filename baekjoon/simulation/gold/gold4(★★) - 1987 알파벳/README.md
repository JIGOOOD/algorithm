### 문제 링크

https://www.acmicpc.net/problem/1987

### 난이도

★★ 아이디어가 잘 안떠오르고, 구현도 그리 쉽진 않음

### 알고리즘 분류

- 백트래킹
- 비트마스킹
- BFS

### 핵심 아이디어: 방문한 알파벳을 비트마스크로 상태 압축 → BFS로 최댓값 탐색

1. 순진한 DFS 백트래킹은 최악의 경우 경우의 수가 너무 많아 Python3에서 TLE → PyPy만 통과
2. 방문한 알파벳 집합을 비트마스크 정수 1개로 표현 (알파벳 26개 → 26비트)
3. 상태를 `(x, y, visited_bitmask)` 튜플로 정의하면 같은 상태를 중복 탐색하지 않음
4. BFS + seen set으로 중복 상태 제거 → Python3에서도 통과
