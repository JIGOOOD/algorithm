### 문제 링크

https://www.acmicpc.net/problem/16724

### 난이도

★ 힌트나 답을 보고 쉽게 풀 수 있음

### 알고리즘 분류

- 그래프
- DFS
- 시뮬레이션

### 핵심 아이디어: 각 칸은 방향이 하나 → Functional Graph → 사이클 개수 = 최소 SAFE 개수

1. 각 칸은 D/U/L/R 방향이 하나뿐이므로, 모든 노드의 출차수(out-degree)가 1인 Functional Graph
2. 이런 그래프에서 모든 경로는 반드시 어떤 사이클로 합류함
3. 사이클 하나당 SAFE 존 1개가 필요 → 사이클 개수가 정답
4. DFS 탐색 시 현재 경로에서 이미 방문한 노드를 재방문하면 → 사이클 발견
5. 경로 상의 노드들은 탐색 완료 후 모두 vis=2로 마킹해 중복 탐색 방지
