### 핵심 아이디어: cctv 방향의 가능한 모든 조합을 4진법으로 하나의 숫자로 표현한다.

cctv 하나가 가질 수 있는 최대 방향 수는 4이므로 4진법 사용

- 모든 cctv 방향 4개 중 하나로 선택 -> 이를 4진수로 바꿔 하나의 숫자로 방향을 선택함
- 4^k = (2^2)^k = 2^(2k)
- pow는 인자를 실수로 받고, 실수로 반환하므로 안됨
