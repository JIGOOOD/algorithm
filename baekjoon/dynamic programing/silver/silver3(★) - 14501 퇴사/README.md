답 봄 (지피티)

### 문제 링크

https://www.acmicpc.net/problem/14501

### 난이도

★ 힌트나 답을 보고 쉽게 풀 수 있음

### 알고리즘 분류

- DP
- 브루트포스

### 핵심 아이디어

1. DP 테이블 정의 $dp[i]$: $i$일차부터 퇴사일까지 벌 수 있는 최대 수익
2. 탐색 방향역순 탐색: 마지막 날($N$일)부터 1일까지 거꾸로 내려오며 계산

   이유: $i$일의 최댓값을 구할 때, 상담이 끝난 시점인 미래($i + T[i]$)의 최댓값을 이미 알고 있어야 하기 때문

3. 핵심 점화식
   - Case A: 오늘 상담이 불가능할 때조건: $i + T[i] > N + 1$ (상담 종료일이 퇴사일을 넘길 때)

     로직: 오늘 상담을 할 수 없으므로, 내일의 최대 수익을 그대로 가져옴

     식: $dp[i] = dp[i + 1]$

   - Case B: 오늘 상담이 가능할 때조건: $i + T[i] \le N + 1$ (퇴사 전까지 상담이 끝날 때)

     선택지
     - 상담을 한다: 오늘 수익($P[i]$) + 상담이 끝난 다음 날의 최대 수익($dp[i + T[i]]$)
     - 상담을 안 한다: 내일의 최대 수익($dp[i + 1]$)식:$dp[i] = \max(P[i] + dp[i + T[i]], dp[i + 1])$
