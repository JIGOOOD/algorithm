### 문제 링크

https://www.acmicpc.net/problem/25682

### 난이도

★ 힌트나 답을 보고 쉽게 풀 수 있음

### 알고리즘 분류

- DP (누적합)

### 핵심 아이디어

1. 체스판 정답 패턴에 맞춰, w로 시작했을 떄와 b로 시작했을 때 틀렸는지 맞았는지를 숫자로 표현하는 배열을 만든다.
   - 틀렸으면 1, 맞았으면 0
   - 어떤 칸 (i, j)에 대해
     - (i + j) % 2 == 0 → 시작 색
     - (i + j) % 2 == 1 → 반대 색
2. 숫자 배열에 2차원 누적합을 사용해 고쳐야하는 최소 값을 센다.
3. 이를 적용한다.
