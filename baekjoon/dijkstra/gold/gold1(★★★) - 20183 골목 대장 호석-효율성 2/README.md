### 알고리즘 분류

- 파라메트릭 서치

### 핵심 아이디어

구해야할 것: 경로에서 사용한 간선 중 최대 비용 (단, 전체 경로 <= C)

1. 이분 탐색으로 최대 간선 비용 X를 정함

   어떤 값 X가 있을 때, 최대 간선 비용 ≤ X 로 제한하면 도착지까지 갈 수 있을까?

   - X가 작으면 → 못 갈 수도 있음
   - X를 늘리면 → 사용할 수 있는 간선이 늘어남 → 갈 가능성 ↑
     -> yes/no -> 이분 탐색 (파라메트릭 서치)

2. 다익스트라로 X 이하인 간선만 사용해서 최단거리가 <= C인 경로가 존재하는지 판단
