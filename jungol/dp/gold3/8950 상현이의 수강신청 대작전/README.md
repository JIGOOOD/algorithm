### ✏️ 알고리즘 풀이

1. 2차원 dp
   dp[i][w] = 최대 w학점일 때, i까지 봤을 때의 최대 선호도 값

2. 1차원 dp

   dp[w] = 최대 학점일 때 가질 수 있는 최대 선호도 값.
   - 왜 뒤에서부터 도는가?
   - 그래야 아직 i를 처리하지 않은 상태라 과목을 하나씩만 처리할 수 있다.
   - 처음부터 돌면 dp[w-c]에 i가 담겨있을 수도 있어, 여러번 담을 수 있게 된다.

   take[i][w] = 과목 i를 처리할 때, w무게에 i를 넣었는지에 대한 값.
